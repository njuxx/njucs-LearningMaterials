# 南京大学计算机学科学习材料汇总
- 该贴主要收集了南京大学计算机学科（计算机科学与技术系、人工智能学院）的教师出版的教材，和部分在线学习资源。
- 我尽可能保证在线学习资源部分是没有版权争议的，也即是老师主动放到公开的视频平台上的。如果材料中存在版权问题，请通过issue与我取得联系。
- 很多绝版书没有被列入教材清单。我对“绝版”的大致定义就是在市面上很难买到。
- 欢迎通过issue补充更多资源。
